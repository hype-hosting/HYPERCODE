This folder contains earlier HYPERCODE prompt tiers that are **no longer actively maintained**. They still work, and you're welcome to use them — but they will not receive updates, bug fixes, or alignment with new HYPERCODE features going forward.

| File | What It Was | Status |
|------|-------------|--------|
| [`essentials.md`](essentials.md) | Compact, token-efficient prompt covering the fundamentals. | Archived |
| [`core.md`](core.md) | Stripped-down, style-agnostic framework. Bring-your-own-voice. | Archived |

## Why Archived?

As of HYPERCODE v1.0, the project consolidated around a single actively maintained prompt — formerly called "Premium," now simply [`hypercode.md`](../hypercode.md) — paired with the [HYPERCODE Preset for SillyTavern](../../../presets/v1.0/HYPERCODE-1.0.json).

Maintaining three parallel tiers fragmented attention and made improvements inconsistent across versions. Going forward, customization is handled through the [Customization Guide](../../../guides/customization.md) (direct edits) and through Outlets (Preset users), rather than through separate prompt tiers.

## Should I Still Use These?

- **Use `essentials.md`** if you specifically need a token-budgeted prompt for a small context window and don't want to manually trim down the full prompt.
- **Use `core.md`** if you want a bare-bones skeleton to build your own framework on top of, independent of HYPERCODE's stylistic defaults.
- **Use [`hypercode.md`](../hypercode.md)** for everything else. It's the recommended starting point.
